# German Learning App

German Learning App is a mobile application developed with React Native and Expo to help users learn the German language in an easy and interactive way.

## Features

- Learn German vocabulary
- Articles and translations
- Interactive exercises and quizzes
- Simple and modern UI
- Mobile-friendly experience
- Fast navigation and performance

## Tech Stack

- React Native
- Expo
- JavaScript / TypeScript
- Expo Router
- NativeWind / Tailwind CSS

## Installation

Clone the repository:

```bash
git clone almand-app
```
Go to the project directory:
```bash
cd almand-app
```
Install dependencies:
```bash
npm install
```
Start the development server:
```bash
npx expo start
```
Project Structure
```bash
almand-app/
│
├── app/
├── assets/
├── components/
├── screens/
├── hooks/
├── constants/
└── package.json
```
Build Android APK
```bash
eas build -p android
```
